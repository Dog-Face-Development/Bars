    You pick a bar, the computer picks a bar and the person (or computer) to pick the last bar wins!
    Written in Python and run through the Python Module (IDLE).

    Features

    • A fun game to play against the computer.
    • Helps children learn and use their problem solving skills and math skills.

    Purpose

    The purpose of this game is to be a quick and easy game to play against the computer whenever you have spare time or are just looking for something to do.

    Installation

    If you have any problems installing the software please contact us over at the Issues page

    Through The Python Module

    • Download the Python module if you haven’t already: Download Link, and install it,
    • Extract files from your GitHub Download,
    • Open the nim.py file in the Python Module (IDLE),
    • Select Run>Run Module, or press F5 in the Edit Window of IDLE,
    • And the app will open for you to use!

    Creating and Executable File Through cx_Freeze (Python 3.4 and lower and Windows Only)

    • Download cx_Freeze if you haven’t already: Download Link, and install it,
    • Extract the files from your GitHub download to this location on your computer: C:\Users\YOUR_USERNAME,
    • Open Command Prompt and type the following command:
    Python setup.py bdist_msi,
    • Run the .msi file that the process creates in a dist sub-directory to create an executable,
    • Run the executable and the app will open for you to use!

    Game Play Instructions

    The object of the game is to be the person to select the last group of bars.

    To start the game, press the space bar after the game has loaded.

    Then click the right most bar from the group of bars that you want to select.

    Then wait for the computer to select some bars.

    Keep repeating the above two steps until you or the computer selects the last group of bars. The person (or computer) to select the last group wins!
